# ArraskaWeb
Arraska (Arrasateko Skate Taldea // Arrasate's Skate Team) is a non-profit group whose aim is encorage skate on the Basque Country.

- Main events are:
    * Skate championships
    * Skate 101 courses
    * Exhibitions
    
This web would provide a place to meet.
